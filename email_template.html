This week's talk is titled 'Information Security - Cyber "vs" Physical' . We know we need to store data somehow so we'll compare physical and cyber information security and how they can actually be integral to each other. The talk will cover a bit of a general look at physical security and how it compares to aspects of cybersecurity, the intersection between them (physical protection of server rooms or key card systems protecting physical stores for example), the different considerations for each of them and where each might be most appropriate.
