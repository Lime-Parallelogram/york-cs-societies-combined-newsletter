We have our first Game Jam for 2025, beginning tonight 19:00 in PZA/023. This is a week-long jam so will be ending on Monday the 24th. Tonight we will play some Jackbox games then vote on a theme for the jam (and people can form teams if they want). Anyone can come along a take part regardless of game dev experience. We've had people make great games using Scratch before or people who have only done drawings.

On the 24th we will do the Game Jam Showcase where people can showcase what they've made and eat free pizza üçï (our AGM will take place on the day as well). You're more than welcome to come along if you haven't made a game.

In addition, on Tuesday, we'll be doing a 3D Unity Workshop 5:30-7:30 in the Creativity Lab (Fairhurst Library). It will be focused on 3D level design and it's being run by Ross Fifield who has lots of experience teaching game development. This is aimed at beginners/intermediate game developers. You don't need to have used Unity before but we do ask that you have some experience with game dev. It will take place in a computer lab which has PCs with Unity installed but you can bring your own laptops if you want to. Please make sure you have a GitHub account set up beforehand so you can download resources during the workshop
